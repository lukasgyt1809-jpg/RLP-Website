# Real Life Project - FiveM Server Website

Eine moderne und ansprechende Website für den Real Life Project FiveM Server mit umfangreichen Informationen und Discord-Integration.

## 🎮 Features

- **Responsive Design**: Funktioniert perfekt auf Desktop, Tablet und Smartphone
- **Modern UI/UX**: Animierte Elemente und interaktive Components
- **Discord Integration**: Direkter Link zum Discord Server mit Widget
- **Schnelle Performance**: Optimiert für schnelle Ladevorgänge
- **SEO Optimiert**: Strukturierter Aufbau für bessere Suchmaschinen-Rankings

## 📁 Projektstruktur

```
RLP-Website/
├── index.html       # Hauptseite mit allen Inhalten
├── styles.css       # Alle CSS-Styles und Animationen
├── script.js        # JavaScript für Interaktivität
└── README.md        # Dokumentation
```

## 🔧 Installation & Setup

### Lokal
1. Repository klonen
2. `index.html` im Browser öffnen
3. Website wird lokal angezeigt

### Deployment

#### GitHub Pages
1. Repository Settings öffnen
2. "Pages" wählen
3. Source: main branch
4. Website wird automatisch unter `username.github.io/RLP-Website` gehostet

#### Andere Hosting-Provider
1. Files auf den Server hochladen
2. Domain konfigurieren
3. Website ist live

## ⚙️ Konfiguration

### Discord Link anpassen
In `index.html` den Discord Link überall ersetzen:
```html
https://discord.gg/pxBqrynn52
```

### Discord Widget aktivieren
In `index.html` Line ~270:
```html
<iframe src="https://discord.com/widget?id=YOUR_DISCORD_SERVER_ID&theme=dark" ...></iframe>
```

Ersetze `YOUR_DISCORD_SERVER_ID` mit deiner Server ID.

### Inhalte anpassen
Alle Texte können direkt in `index.html` bearbeitet werden.
